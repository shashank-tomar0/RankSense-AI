2. Project Overview

TalentScout AI is a smart recruitment tool that helps companies find the best candidates in seconds. Instead of just looking for keywords like old systems, our tool actually understands the resume content.

We built this because hiring is slow and people try to "cheat" the system with hidden keywords. TalentScout is designed to be Fast, Secure, and Fair for everyone—giving high-potential freshers a real chance to shine against experienced pros.

3. Key Features

A. Safety & Fairness

  • Cheat Detection: Finds "hidden keywords" or invisible text used to trick hiring tools.
  • Fair Scoring: Automatically shifts focus to projects and marks for students/freshers.

B. Deep AI Insights

  • Smart Summaries: Quick AI reports on candidate fit.
  • Interview Pilot: Suggests 5 custom questions for each candidate.
  • Battle Royale: Side-by-side comparative analysis between two candidates.

C. High-Speed Processing

  • Instant Analysis: Sub-second resume analysis using specialized hardware.
  • Live Logs: Real-time visibility into the system's thinking process.

D. Real-World Verification

  • GitHub Trust Chain: Verifies skill claims by checking real GitHub activity.
  • Smart Data Extraction: OCR technology to read even scanned or low-quality files.

E. Recruiting & Diversity Tools

  • One-Click Outreach: AI-personalized email drafting for candidates.
  • Anti-Bias Mode: Hides PII to focus entirely on skills and experience.

4. How the Score is Calculated

  1. Security Check: First, we check if the file is "clean".
  2. Extraction: We pull out 12 key factors like Skills, Internships, Projects, and GPA.
  3. Adjustment: Weights are shifted for Freshers to ensure fairness.
  4. Verification: Cross-checking claims with GitHub activity.
  5. Final Result: A score out of 100 based on all factors.
